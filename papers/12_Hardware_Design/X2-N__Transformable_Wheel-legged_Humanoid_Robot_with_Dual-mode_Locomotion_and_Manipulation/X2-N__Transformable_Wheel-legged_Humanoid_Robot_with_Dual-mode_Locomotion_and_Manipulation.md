# X2-N: A Transformable Wheel-legged Humanoid Robot with Dual-mode Locomotion and Manipulation

**移动平台往往「能走的不灵巧、灵巧的走不快」：人形上肢灵活、能稳定站立操作，但腿走起来低效又怕冲击；轮足机器人移动高效，却缺上肢灵巧度和稳定操作能力。X2-N 用一台「可形变」机器人把两者合一：它能在「足式人形形态」和「轮足形态」之间通过关节重构无缝互变（约 1 秒），关键设计是「关节复用」——把传统髋偏航关节挪到腿末端，与一个直驱轮毂电机合二为一，既当轮子又当转向关节，从而不为变形多加冗余电机。配套一个面向该形态定制的 RL 全身控制框架，统一管轮足滑行、足式行走、形态切换与上肢操作。约 28 kg、1.1 m，真机演示了动态滑行、上下楼梯、坡道送包裹与抗推扰。**

## 🎯 一句话总结

现有移动平台普遍在「移动效率」与「操作灵巧」之间二选一：

- **人形**：上肢灵活、站立稳，但**足式行走低效、易受冲击**；
- **轮足**：移动高效，但**缺上肢灵巧度、操作稳定性差**。

X2-N 想要「鱼与熊掌兼得」——做一台能在**足式人形**与**轮足**两种形态间无缝切换的机器人：需要快时切到轮足高效巡航，需要灵巧操作/越障时切回足式人形，并用一套 RL 全身控制把所有模态统一管起来。

## ❓ 论文要解决什么问题？

**问题陈述**：单一形态的移动机器人难以同时满足「高效移动 + 稳定灵巧操作」。

1. **人形**腿部行走能耗高、关节峰值载荷大、对冲击敏感；
2. **轮足**移动省力，却没有人形那样的上肢灵巧与站立操作能力；
3. 若简单地「轮 + 腿都装上」，往往要**为形态切换额外堆电机**，增重又复杂。

**核心问题**：

> 能否设计一台**形态可变**的机器人，在「足式人形」与「轮足」之间**快速无缝切换**，且**不增加冗余执行器**，并用**统一的控制框架**驾驭两种形态下的移动、切换与操作？

## 🔧 方法拆解：机构设计 + RL 控制框架

### 1. 关节复用（Joint Reuse）与变形机构

- **整机**：约 **28 kg、1.1 m**；足式人形形态 **21 DoF**，轮足形态 **17 DoF**。
- **核心招式「关节复用」**：把传统的**髋偏航（hip-yaw）关节挪到腿末端**，并集成一个**直驱轮毂电机**——它**既是轮子、又是转向关节**，因此变形不需要新增专用电机。
- **腿部拓扑**：足式 6-DoF 腿（3-DoF 髋 + 1-DoF 膝 + 2-DoF 踝）；轮足下退化为 4-DoF（髋 pitch/roll + 膝 + 轮转向）。
- **变形过程**：约 **1 秒**完成——主动控制踝 roll 关节重定向脚踝姿态，并用**含卡扣/多接触面的锁定机构**在轮足模式下把轮座刚性锁死。工作空间分析显示其腿部凸包覆盖约为常规人形腿的 **87.8%**。
- **上肢**：模块化可换臂，**4-DoF 轻量**或 **7-DoF 灵巧**（带夹爪）；躯干为金属-聚合物混合结构，内嵌散热与前向相机。

### 2. 定制执行器（QDD）

四款自研准直驱电机：**R90**（峰值 120 N·m / 990 g）、**R57**（30 N·m / 370 g）、**R52**（20 N·m / 360 g）、**R52-U**（40 N·m / 410 g）；行星减速、电流-力矩近线性，无需专门力矩传感器。算力为 Rockchip RK3588（NanoPi），1 kHz CAN-FD 通信，约 1 小时续航。

### 3. RL + 模型混合控制

- **RL 移动（PPO，Isaac Gym 训练 → MuJoCo 验证）**，三个独立智能体：
  1. **轮足智能体**：带「**离心力感知**」奖励 + 身体侧倾策略，保证高速转弯不失稳；
  2. **足式智能体**：用 **AMASS 动捕**做步态模仿奖励，行走更自然；
  3. **混合智能体**：三阶段课程学习「**类滑冰**」滑行，克服单点接触的不稳定。
- **变形控制器**：对两个切换方向分别设计，用预计算关节轨迹 + 横滚/俯仰稳定奖励维持平衡；
- **模型化 WBC**：基于 Newton-Euler 动力学 + 逆运动学做末端轨迹规划，实现稳定 loco-manipulation。
- 推理 **50 Hz**、执行器控制 **1 kHz**。

## 💡 核心贡献

1. **可形变轮足人形本体**：足式人形 ↔ 轮足两形态**约 1 秒无缝互变**，兼顾移动效率与上肢灵巧操作；
2. **关节复用机构**：髋偏航关节下移至腿末端并与直驱轮毂电机合一，**变形不增加冗余执行器**，整机轻量（约 28 kg）；
3. **面向形态定制的 RL 全身控制**：三个 PPO 智能体（轮足 / 足式 / 混合滑行）+ 变形控制器 + 模型化 WBC，统一管移动、切换与操作；
4. **真机验证**：动态滑行、上下楼、坡道与楼梯送包裹、推扰恢复、夹爪操作等多任务演示；
5. **效率优势**：同速下轮足模式相比足式具有**更低关节峰值载荷与更低能耗**。

## 🤖 对人形硬件设计的意义

| 方向 | 含义 |
|---|---|
| **形态可变** | 「足式 ↔ 轮足」按需切换，让一台机器人覆盖更广的地形与任务谱 |
| **关节复用** | 用机构巧思而非堆电机来实现变形，是减重/降复杂度的可借鉴范式 |
| **统一控制** | 多模态由一套 RL + 模型混合框架管理，降低多形态系统的控制工程量 |
| **loco-manipulation** | 在移动能力之外保留上肢灵巧，贴近「移动 + 作业」的真实落地场景 |

## 🎤 面试参考

**Q：X2-N 解决了什么核心矛盾？**
A：移动平台常「能走的不灵巧、灵巧的走不快」。人形上肢灵活但足式行走低效易受冲击；轮足移动高效却缺上肢灵巧与稳定操作。X2-N 用可形变设计，在足式人形与轮足之间约 1 秒互变，按需取长补短。

**Q：「关节复用」具体省掉了什么？**
A：把传统髋偏航关节挪到腿末端，与一个直驱轮毂电机合并——同一电机在轮足模式当轮子兼转向关节，在足式模式提供腿部自由度。这样形态切换无需额外加专用电机，整机更轻（约 28 kg）。

**Q：变形是怎么做到约 1 秒且稳的？**
A：主动控制踝 roll 关节重定向脚踝姿态，配合含卡扣/多接触面的锁定机构在轮足模式刚性锁死轮座；变形控制器用预计算关节轨迹 + 横滚/俯仰稳定奖励维持平衡。

**Q：为什么用三个独立 RL 智能体而不是一个？**
A：轮足高速转弯、足式自然步态、单点接触的滑行各有不同的动力学与奖励诉求。分别训练（轮足离心力感知 + 侧倾、足式 AMASS 模仿、混合滑行用课程学习）再加模型化 WBC，比单策略更易稳定收敛。
